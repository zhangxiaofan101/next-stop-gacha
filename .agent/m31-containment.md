## D. 已核对并排除的易误判

- `湖州·莫干山` / `湖州·南浔` / `安吉`：名称已用后缀收窄，彼此是兄弟目的地。
- `舟山·普陀山` / `舟山·桃花岛朱家尖` / `东极岛` / `嵊泗·枸杞岛`：共享舟山定位不等于普陀山岛包含其他岛。
- `台州·临海` / `温岭·石塘` / `天台山`，`温州·楠溪江` / `温州·洞头岛`，`泰州·兴化垛田` / `泰州·溱湖溱潼`：均为已收窄的兄弟卡。
- `黔东南·肇兴侗寨` / `黔东南·西江千户苗寨` / `黔东南·从江加榜梯田`：共享州名前缀本身不构成包含；只有 C5 的方案内容重叠。
- 呼伦贝尔与阿尔山不是行政包含；承德与乌兰布统跨省区相邻；五台山与忻州·雁门关是同一地级范围下的不同县域目的地。
- 独库公路只穿过伊犁卡中的那拉提等局部，不能据此判定独库与整张伊犁卡互相包含。
- `武功山` 与 `宜春·明月山` 有广义山系名称歧义，但两卡旅游范围分别是萍乡金顶线与宜春明月山—温汤，维持并列；可把前者改名「萍乡·武功山（金顶）」消歧。
